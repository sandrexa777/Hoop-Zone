# Hoop-Zone

A responsive basketball website (coaching + informative) built with HTML & CSS only.

## Pages
- index.html — Home
- fundamentals.html — Fundamentals & drills
- positions.html — The five positions
- contact.html — Join form (with required fields)

## Features
- Fully responsive (mobile menu, fluid layouts)
- Two Google Fonts: Bebas Neue + Inter
- Pure HTML/CSS, no JavaScript
